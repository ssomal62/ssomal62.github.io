group 용량 제한 #

  • 네이버는 한 사람이 3개까지 계정 생성 가능

  • 네이버 각 메일 용량 5G (네이버 홈 )

  • 네이버 마이박스 계정 총 용량 30G

  • 구글은 계정마다 15G

  • 한 사람의 계성생성 제한 없음
