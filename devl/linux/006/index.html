권한 (permision) - 115p #

# 타임+권한   #하드링크  #소유주/소유그룹                     
-rw-r--r--      1      root  root      193 10월 24 10:08 a1  #파일은 하얀색
drwxr-xr-x      2      root  root     4.0K 10월 24 10:08 d1/ #디렉토리 파란색
  • drwxr-xr-x : 권한은
    • 3개씩 끊어서 보면됨 -> d/rwx/r-x/r-x
      #user 권한       #group 권한      # other 권한
d        rwx            r-x               r-x
  • r(=4) : 읽기 권한
  • w(=2) : 생성, 수정 및 삭제 권한
  • x(=1) : 실행 권한
      #user 권한       #group 권한      # other 권한
d        rwx            r-x               r--
      #(r+w+x) 7      #(r+x) 5          #(r) 4
  • 디렉토리의 실행(x)권한? 디렉토리로 진입 할 수 있는 권한
