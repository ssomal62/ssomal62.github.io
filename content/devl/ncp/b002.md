#### 접속


```shell
# 1) 업데이트
:~# apt-get update
# 2) MYSQL 설치
:~# apt-get install mysql-client
```

* 설치 전 update를 해준다.
* mysql 설치


```shell
# 접속하는 방법
:~# mysql -h db-jbnp5-kr.vpc-pub-cdb.ntruss.com -u study -p
```

* `-h`: 호스트 주소. 퍼블릭 도메인 입력
