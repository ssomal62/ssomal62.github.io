## UNION
- 두 개의 SQL의 결과를 결합 ->  중복된 행을 제거
- 결과 집합에서 중복된 행을 하나로 처리.

```sql
SELECT column1, column2 FROM table1 
UNION 
SELECT column1, column2 FROM table2;
```


## UNION ALL
- 두 개의 SQL의 결과를 결합 ->  중복된 행을 제거하지 않음.
- UNION보다  결과가 빠를 수 있음.

```sql
SELECT column1, column2 FROM table1 
UNION ALL 
SELECT column1, column2 FROM table2;
```
